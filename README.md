**Git Flow:** `main` → `develop` → `feature/*`

## Ветки
- `main` — только релиз (защищён)
- `develop` — основная разработка
- `feature/*` — задачи (PR → develop)

## Как работать
```bash
git checkout develop
git pull
git checkout -b feature/твоя-задача
# код → commit → push → PR в develop
